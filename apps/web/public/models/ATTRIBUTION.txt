Most 3D craft preview models in this folder were copied from the Betaflight Configurator
project (see the original-assets exception below):
https://github.com/betaflight/betaflight-configurator

Source location:
- resources/models/*.gltf

Upstream license:
- GPL-3.0

These assets are kept here so the browser Setup and Receiver experiences can provide a real
rotating craft preview similar to Betaflight's configurator instead of a placeholder mockup.

Retain upstream license obligations when redistributing these files.

----------------------------------------------------------------------

Plane / VTOL meshes — bixler.gltf, alti.gltf

These two are the ArduPilot fixed-wing + VTOL craft previews, copied from the
ArduPilot Configurator project:
https://github.com/ArduPilot/ArduConfigurator

Source location:
- resources/models/bixler.gltf                   -> bixler.gltf  (fixed-wing)
- resources/models/alti-transition3.propped.gltf -> alti.gltf    (VTOL / tiltrotor)

Per that project's resources/models/gltf-notes.txt its aircraft meshes are sourced
from free, public-domain / unlicensed models ("we deliberately do NOT have any
licensed models"); the project itself is GPL-3.0. Retain that provenance when
redistributing.

----------------------------------------------------------------------

Exception — plane.gltf, rover.gltf, sub.gltf

These three are NOT from Betaflight. They are original models authored for this project
(Betaflight ships multirotors only, so ArduPlane / ArduRover / ArduSub had no preview):

- plane.gltf — fixed-wing, generated by scripts/generate-plane-gltf.mjs
- rover.gltf — ground rover, generated by scripts/generate-rover-gltf.mjs
- sub.gltf   — underwater ROV, generated by scripts/generate-sub-gltf.mjs

Each is generated deterministically by its script and released under CC0-1.0 (public
domain dedication); no upstream license obligations attach to them.
