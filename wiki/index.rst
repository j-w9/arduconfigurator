ArduConfigurator Wiki
=====================

ArduConfigurator is a browser-first configurator for ArduPilot — a setup- and
configuration-first tool aimed at ArduCopter FPV workflows (and, increasingly,
Plane, Rover, and Sub). It runs entirely in your browser over Web Serial, with a
thin desktop shell that reuses the same app.

This wiki mirrors the structure of the `ArduPilot wiki <https://ardupilot.org/>`__
and the order of the app itself: connect, run the guided setup, then dive into
each tab. If you are new, start with :doc:`introduction` and
:doc:`getting-connected`, then follow :doc:`guided-setup`.

.. note::

   ArduConfigurator complements the ArduPilot wiki — it does not replace it. For
   firmware concepts, flight theory, and tuning background, the
   `ArduPilot documentation <https://ardupilot.org/>`__ remains the canonical
   reference. This wiki documents the **configurator**: what each surface does
   and how to use it.
